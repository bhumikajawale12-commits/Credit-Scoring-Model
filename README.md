# Credit Scoring Model

## Objective
To predict whether a loan applicant is creditworthy using machine learning classification.

## Dataset
German Credit Dataset

## Algorithms Used
- Logistic Regression
- Decision Tree
- Random Forest

## Tools Used
Python, Pandas, Scikit-learn, VS Code

## How to Run
pip install pandas numpy scikit-learn joblib  
python task1.py

## Output
Model evaluation metrics printed in terminal.
